Detailed Description

template<typename T, template< typename > typename K>
class DataFlow::Consolidator< T, K >

Consolidates multiple sources into a single one.

The template parameter K is the method with which the source consolidation is performed. The type of consolidation must match the type of the consolidator. See the Consolidation namespace for implemented methods.
