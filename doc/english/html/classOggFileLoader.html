Detailed Description

template<typename T, typename K>
class OggFileLoader< T, K >

Class for reading OGG encoded files. Template type K is the decoder this instance will use.

Usage:

  1. open file with open()
  2. initialize with init()
  3. the loader instance is ready to be read from with get(amount)
