Detailed Description

template<typename T>
class Mixer::PortAudioBackend< T >

An endpoint with PortAudio backend

This class utilizes the cross-platform audio backend PortAudio to play data from an iSource.
The init() function must be called before starting playback with the start() function. The source must contain the audio data already multiplexed.
