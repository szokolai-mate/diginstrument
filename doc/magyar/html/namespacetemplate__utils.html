◆ extract_inner

template<typename T >
using template_utils::extract_inner = typedef typename extract_inner_base<T>::type

This will extract the template parameter of template parameter T For example, if you have A with <typename t>=""> and B with <typename k>="">, and make an A<B<float>>, then in A extract_inner<T> will evaluate to K or float in this case.
