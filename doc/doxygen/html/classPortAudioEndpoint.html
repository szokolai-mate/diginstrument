class PortAudioBackend< T >

PortAudio backendet használó végpont

Ez az osztály egy multiplatformos audió backendet, a PortAudio-t használja egy iSource objektumból nyert audió adat lejátszására. Használat előtt az init() függvénnyel egy iSource objektumot csatlakoztatunk hozzá, majd a start() fügvénnyel elkezdhetjük a lejátszást. A csatolt objektumból nyert adat multiplexáltnak tekintendő.
